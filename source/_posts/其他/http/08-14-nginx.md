#### Nginx

#### nginx命令

- nginx -s stop // 停止
- nginx -s quit //退出
- nginx -s reload // 重新加载配置文件
- ps aux|gre nginx // 查看nginx进程

/usr/local/etc/nginx/nginx.conf （配置文件路径）
/usr/local/var/www （服务器默认路径）
/usr/local/Cellar/nginx/1.6.2 （貌似是安装路径）
