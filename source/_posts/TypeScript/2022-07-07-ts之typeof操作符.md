### typeof: 用来获取一个变量或对象的类型

```ts
const Person = {
	name: 'John',
	age: '16',
}

type CustomTypeOf = typeof Person  // { type CustomTypeOf = {name: string; age: string;}}
 
const a: CustomTypeOf = {name: 'John', age: '16'}

```
